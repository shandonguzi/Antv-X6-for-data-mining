### 1. Choose to go to different folders according to your computer system

For win：

```bash
manage.exe runserver --noreload
```

For mac：

```bash
./manage runserver --noreload
```

For linux：

```bash
./manage runserver --noreload
```

### 2. Enter the address http://127.0.0.1:8000/ in browser
